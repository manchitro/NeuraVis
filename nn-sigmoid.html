Sigmoid Neurons

Previously, we learnt about perceptrons and how they take binary inputs and produce a binary output.
Sigmoid neurons are almost identical to perceptrons. The only difference is that, sigmoids take any under between 0 and 1 as inputs and outputs the same.

A sigmoid's input and output might look something like the above image. See it is no longer constrained as binary but any floating point number between 0 and 1.
